Projecte final al repte clubs de ciencia Mèxic 2020 Regles i directrius aquí!

Aquest document presenta una proposta d'una pàgina web per poder ser utilitzat per als clubs, si deciden crear-la per aquest mètode per aprendre un poc d'HTML d'una forma fàcil i pràctica, únicament han de modificar el text i les etiquetes que apareixen a l'arxiu index.html que es troba al repositori d'aquesta pàgina web. Aqui tenen una alternativa per crear el seu propi lloc. poden fer copy-page de les línies de codi que apareixen en aquest índex per afegir els elements de la seva pàgina web o bé d'una manera d'exemple per saber quins són els comandaments per afegir elements i utilitzar-los a la seva pròpia pàgina. aquesta pàgina conté elements com a títols, pestanyes deslizables, imatges i vídeos de youtube. Tots els elements audiovisuals tendran que pujar a la carpeta assets/img/ i únicament col·locar la ruta on volem afegir l'element. 10/08/2020.
